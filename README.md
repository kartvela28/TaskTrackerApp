# Task Tracker App  
A simple console-based task tracking application written in C#.  

## ✅ Features  
- Add a new task with description  
- List all tasks, or filter by status  
- Update task description by ID  
- Change task status (e.g., Todo → InProgress → Done)  
- Delete a task by ID  
- JSON-based persistence: tasks are stored on disk between runs  
- Clean architecture: separation of concerns (UI layer, repository/data layer, domain models)  

### Prerequisites  
- [.NET 9.0 SDK](https://dotnet.microsoft.com/en-us/download) (or whichever version you target)  
- A code editor: e.g., Visual Studio Code, Visual Studio, or similar  
- Git (optional) if you want to clone the repository
